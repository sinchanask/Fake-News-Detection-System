# 📌 Fake News Detection System

## 📖 Project Description
This project is a Fake News Detection System built using Machine Learning and Natural Language Processing (NLP) techniques.  
It classifies news articles as Fake or Real based on their textual content.

---

## 🎯 Objectives
- Classify news as Fake or Real  
- Reduce the spread of misinformation  
- Build an automated news verification system  
- Apply NLP techniques for text analysis  

---

## ⚙️ Technologies Used

### 💻 Programming Language
- Python  

### 📚 Libraries
- Pandas  
- NumPy  
- Scikit-learn  
- NLTK  
- Matplotlib  
- Seaborn  

### 🛠️ Tools
-  Google Colab  
- CSV Dataset  

---

## 🔄 Workflow
1. Data Collection  
2. Data Preprocessing (cleaning, tokenization, stopword removal, stemming)  
3. Feature Extraction using TF-IDF  
4. Train-Test Split  
5. Model Building (Naive Bayes, Logistic Regression, Random Forest)  
6. Model Evaluation (Accuracy, Precision, Recall, F1-score)  
7. Prediction System Development  

---

## 📊 Results
- Successfully classified news as Fake or Real  
- Achieved good accuracy using ML models  
- Compared model performance using graphs  
- Developed a basic working prediction system
